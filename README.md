Food Delivery Management System (SQL, DBMS)

This project implements a relational database system for managing
restaurants, customers, orders, and delivery operations.

Features
• Normalized relational schema
• Stored procedures for order processing
• Triggers for maintaining data consistency
• SQL queries for order analytics

Technologies
• SQL
• PL/SQL
• DBMS concepts (Normalization, Triggers, Procedures, Cursors)
